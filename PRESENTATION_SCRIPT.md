## **SLIDE 1: Introduction (30 seconds)**

> "Hello everyone! Today I'm going to present **Civic Pressure** ‚Äî a modern web platform that bridges the gap between citizens and local government.

> Have you ever noticed a broken road, garbage pile, or water supply issue in your neighborhood and wished you could easily report it? That's exactly the problem Civic Pressure solves.

## **SLIDE 2: Key Features (45 seconds)**

> "Let me walk you through the key features of our platform:

> **1. Easy Complaint Reporting** ‚Äî Citizens can submit complaints with detailed descriptions, select categories like roads, water supply, garbage, electricity, and pin the exact location on an interactive map.

> **2. AI-Powered Intelligence** ‚Äî This is where our app stands out. We've integrated:
>    - **Google Gemini AI** to automatically improve complaint descriptions, making them more professional and actionable for government officials
>    - **Semantic Duplicate Detection** using TensorFlow.js that identifies similar complaints already submitted, preventing redundancy

> **3. Community Engagement** ‚Äî Users can vote on complaints to increase priority, support similar complaints, and get real-time notifications on status updates.

> **4. User Dashboard** ‚Äî Track all your complaints, see their status, and manage your profile."

---

## **SLIDE 3: Tech Stack (30 seconds)**

> "The application is built using modern, scalable technologies:

> **Frontend:** React 18 with TypeScript, Tailwind CSS for styling, and Google Maps/Leaflet for interactive mapping

> **Backend:** Node.js with Express.js, handling REST APIs with Firebase Admin SDK

> **Database:** Firebase Firestore for real-time data, Firebase Storage for images, and Firebase Authentication for secure login

> **AI/ML:** Google Gemini API for text improvement and TensorFlow.js with Universal Sentence Encoder for semantic similarity matching"
